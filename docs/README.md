# Документационный контур проекта

Этот раздел собирает весь контекст по разработке MVP личного кабинета клиента по экологии. При добавлении новых материалов придерживаемся текущей структуры, чтобы каждая следующая сессия ИИ мгновенно попадала в нужный контекст.

- `documentation-plan.md` — детальная дорожная карта подготовки документации по функциональным блокам MVP.
- `assistant-guidelines.md` — правила поведения для ИИ при работе над проектом.
- `completed/` — журнал зафиксированных результатов (каждый завершённый пункт фиксируем отдельной записью).
- `documentation-todo.md` — сводная таблица открытых вопросов и TODO по всем документам.
- `development-readiness.md` — аудит готовности документации к старту разработки.
- `architecture/adr/` — решения по стеку и инфраструктуре (ADR 0001, ADR 0002).

При появлении новых файлов или подпроектов обязательно обновляем этот индекс.
