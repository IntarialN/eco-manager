# Чат-виджет и выравнивание макета

**Контекст**
- Требование 100vh (шапка сверху, контент, футер снизу) и кнопка, открывающая чат.
- Ссылка на `docs/features/chat-support.md` (раздел UI/UX) и `docs/documentation-todo.md` (пункт «Веб-чат и обратный звонок»).

**Что сделано**
1. Обновлён основной layout `yii-app/views/layouts/main.php`: навигация/контент/футер распределены в пределах 100vh, добавлен плавающий виджет с формой (имя, контакт, сообщение, телефон) и JS-клиент для POST `/chat/session` + `/chat/{id}/callback`.
2. В `yii-app/web/css/site.css` добавлены стили для новой сетки (flex контейнеры, `main-container`, фиксированный футер) и оформления чат-панели/кнопки, включая адаптив для мобильных экранов.
3. Документация: `docs/features/chat-support.md` и `docs/documentation-todo.md` обновлены, чтобы отразить реализацию базового веб-виджета и оставшиеся задачи (WebSocket, операторский UI).
