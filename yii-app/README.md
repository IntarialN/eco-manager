# Eco Manager — Yii2 Prototype

Шаблон личного кабинета клиента, реализованный на Yii2 (basic template). Включены основные блоки MVP:

1. **Карта требований**
2. **Артефакты/Хранилище**
3. **Календарь событий**
4. **Риски**
5. **Договоры/счета/акты**

## Локальный запуск

1. Установите зависимости:
   ```bash
   composer install
   ```
2. Примените миграции (создастся SQLite база `data/eco_manager.db`):
   ```bash
   php yii migrate
   ```
3. Запустите встроенный сервер:
   ```bash
   php yii serve --docroot=@app/web --port=8080
   ```
4. Перейдите в браузере по адресу [http://localhost:8080](http://localhost:8080).

## Структура

- `controllers/ClientController.php` — вывод данных клиента и основных блоков.
- `models/` — ActiveRecord-модели (Client, Requirement, Document, CalendarEvent и др.).
- `migrations/` — создание схемы и примерных данных для одного клиента.
- `views/client/` — вкладки интерфейса личного кабинета.
- `web/uploads/` — папка для артефактов (можно добавить реальные файлы).
